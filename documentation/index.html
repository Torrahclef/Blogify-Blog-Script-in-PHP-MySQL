Installation

To install the Blogify follow these steps:

Upload the files

→ This script requires Apache Server, PHP 5.3+ and MySQL 5.0+ to run.

→ Before install Blogify, make sure you create a New Database along with a user

→ After unzip Blogify Files, you will find all Folders and files

→ Login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.

→ Locate the domain folder you wish to upload your Blogify files to your server. You are going to upload all files.

→ Once the files are done uploading, No coding or design skills required. Just go to your new Blogify website link and installer will run automatically http://yourdomain.com/path_to_script.

→ After you have successfully installed your new blog, we advice you delete the install folder from your server for security reasons.

Access the site Admin area

When the application is installed, an administrator is created:

Admin Area Options

 

Dashboard: This is the admin's home page.

Manage Articles: Section where you can create new article, and manage all articles

Manage Categories: Here you create new article categories or manage all categories

Youtube Video: Section where you post daily Videos from youtube on your blog

Daily Quote: Here you update daily quotes on your blog

Manage Admin: Section where you can create admin and also manage admins

Settings: To configure the more general aspects of the application.
